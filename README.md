# handle-data-format
How to handle different data format in python

## Setup

* Create virtual environment
    ```bash
    python3 -m venv .venv
    ```
* Install requirements
    ```bash
    pip install -r requirements.txt
    ```
  
* put some csv file in data folder and name it data.csv
* run the main script
    ```bash
    python main.py
    ```
